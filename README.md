# Perspective AI: Strategic Financial Intelligence

An advanced, minimal financial reporting application that leverages Google's Gemini 3.1 Pro to synthesize deep perspectives across history, present, and future trajectories.

## 🚀 Features
- **Advanced AI Analysis**: Generates comprehensive reports covering businesses, jobs, stocks, market sectors, AI era economics, global politics, climate impact, and human psychology.
- **Strategic Action Plans**: Provides structured, numbered steps for strategic decision-making.
- **Minimalist Technical Design**: A high-contrast, dark-themed interface inspired by professional data dashboards.
- **PDF Export**: Download polished, presentation-ready PDF reports.
- **Interactive Perspectives**: Tabbed interface to explore different time horizons and strategic dimensions.

## 💻 Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS v4, Framer Motion
- **UI Components**: shadcn/ui (Radix UI primitives)
- **AI Integration**: `@google/genai` (Gemini 3.1 Pro)
- **Export**: `jspdf`, `html2canvas`

## 🛠️ Setup & Installation
1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file based on `.env.example` and add your Gemini API key:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```
4. Start the development server: `npm run dev`
